# CRM Platform

A production-ready CRM Platform built with React, TypeScript, and modern frontend technologies.

---

## Features

- Authentication
- Role Based Access
- Dashboard
- Customer Management
- Lead Management
- Deal Pipeline
- Task Management
- Reports & Analytics
- Notifications
- Dark Mode
- Responsive Design

---

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
- Redux Toolkit
- Tailwind CSS
- Axios
- React Query
- React Hook Form
- Zod
- Recharts
- Jest

---

## Installation

```bash
git clone https://github.com/yourusername/crm-platform.git

cd crm-platform

npm install

npm run dev
```
